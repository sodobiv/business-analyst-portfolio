# Анализ трудозатрат и выручки (дашборд эффективности)

## 📌 Задача
Создать инструмент для анализа распределения ресурсов, оценки эффективности проектов и управления загрузкой сотрудников.

## 🛠 Инструменты
- **Power BI** — построение дашборда
- **Excel / Power Query** — подготовка и обработка данных
- **DAX** — расчёт метрик

## 🔍 Подход
1. Собраны и структурированы данные по трудозатратам, выручке и ролям участников проектов.
2. Построен дашборд с разрезами:
   - По продуктовым направлениям
   - По функциям сотрудников (эксперты, интеграторы, координаторы и др.)
   - По уровням специалистов
   - По проектам и месяцам
3. Рассчитаны ключевые метрики эффективности.

## 📊 Результаты
- Проанализированы данные по **100+ сотрудникам**, **10+ проектам**, **8 направлениям**
- Рассчитаны:
  - Общие и средние трудозатраты по направлениям
  - Выручка по направлениям и проектам
  - Количество уникальных сотрудников по функциям
  - Активность сотрудников по месяцам

## 💡 Ценность для бизнеса
Руководство получило инструмент для:
- Оценки рентабельности направлений
- Оптимизации распределения ресурсов
- Обоснования управленческих решений (например, перераспределение экспертов)
- Планирования загрузки сотрудников
