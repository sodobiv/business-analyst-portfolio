# Научная статья по графам знаний

## 📌 Тема
Исследование применения онтологий (графов знаний) для моделирования карт знаний и управления данными.

## 🛠 Методология
- Теоретический анализ подходов к управлению знаниями
- Построение модели графа знаний (онтологии)
- Применение модели для структурирования экспертизы в организации

## 📄 Публикация
**Иванова С.Д.** *Граф знаний (онтологии) как информационная модель карты знаний* // Технологии искусственного интеллекта в науке и образовании: Материалы конференции.

## 📊 Результаты
- Разработана модель представления знаний на основе графов
- Показано, как онтологии помогают структурировать информацию о компетенциях и экспертизе

## 💡 Ценность для бизнеса
Методология может быть использована как база для:
- Создания систем управления знаниями в компаниях
- Онбординга новых сотрудников (карты знаний)
- Поиска экспертов внутри организации
- Построения маршрутов обучения
