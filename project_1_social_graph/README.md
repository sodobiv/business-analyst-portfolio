# Картирование социального графа

## Задача
Проанализировать коммуникации в команде из 100+ сотрудников, выявить скрытые связи и точки роста для улучшения взаимодействия.

## Инструменты
- Gephi (построение графа)
- Excel (подготовка данных)
- Метрики центральности

## Результат
- Выявлено 5 кластеров со связанностью 12–26%
- Определено 5 «брокеров знаний» для улучшения коммуникаций
