# Картирование знаний и дашборд компетенций

## 📌 Задача
Разработать методологию картирования знаний и создать интерактивный дашборд для визуализации экспертизы сотрудников по направлениям.

## 🛠 Инструменты
- **Power BI** — построение дашборда
- **Excel / Power Query** — подготовка данных
- **Gephi** — визуализация социальных связей экспертов
- **Таксономия знаний** — классификация компетенций

## 🔍 Подход
1. Разработана таксономия знаний (структура компетенций)
2. Создана анкета для сбора данных о навыках сотрудников
3. Спроектирована модель данных и связи между сущностями
4. Построен интерактивный дашборд с фильтрами по подразделениям, специализациям и уровням экспертизы

## 📊 Результаты
- В дашборд загружены данные по **60+ сотрудникам** и **5+ направлениям**
- Выявлены зоны с дефицитом экспертизы (в одном направлении **33 сотрудника** имеют уровень знаний не выше 3 из 5)
- Определены сотрудники, готовые развиваться в новых направлениях

## 💡 Ценность для бизнеса
Дашборд позволяет:
- Находить экспертов под задачи за **5 минут**
- Видеть «слепые зоны» в компетенциях команды
- Планировать обучение и развитие сотрудников
- Формировать проектные команды на основе данных
